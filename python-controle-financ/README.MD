# Controle Financeiro Pessoal (GUI)

Aplicação desenvolvida em Python utilizando Tkinter, com o objetivo de praticar
lógica de programação e desenvolvimento de interfaces gráficas.

## Funcionalidades
- Registro de entradas e saídas financeiras
- Listagem das movimentações
- Cálculo automático do saldo
- Persistência de dados em arquivo JSON

## Tecnologias
- Python 3
- Tkinter
- JSON

## Objetivo
Projeto com foco em nível júnior para reforçar conceitos de lógica,
organização de código e interfaces gráficas.
